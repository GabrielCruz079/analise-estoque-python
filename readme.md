# 📦 Análise de Estoque

Projeto em Python para análise de dados de estoque e financeiro.

## 🚀 O que o projeto faz
- Calcula valor total em estoque
- Identifica produtos
- Gera gráfico de quantidade

## 🛠️ Tecnologias
- Python
- Pandas
- Matplotlib

## ▶️ Como rodar
```bash
pip install pandas matplotlib
python analise.py
